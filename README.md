这是我的个人博客网站，用于分享技术文章和生活感悟。网站采用纯静态设计，通过 GitHub Pages 部署。

## 🌟 特性

- 响应式设计，完美适配各种设备
- 现代化 UI 设计，提供良好的用户体验
- 支持博客文章展示
- 集成社交媒体链接
- 基于 GitHub Pages 自动部署

## 🛠️ 技术栈

- HTML5
- CSS3 (使用现代特性如 Flexbox, Grid, CSS Variables)
- JavaScript
- GitHub Actions (自动部署)
- Jekyll (静态站点生成)
